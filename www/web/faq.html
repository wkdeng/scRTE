Tutorial

1.     Search options

a.     Search single term:

                                               i.     Search by RTE

In this option, scARE will search RTEs with any of Class, Family or Name matching or partial matching keyword. Result will be a table of RTEs. By checking “I’m feeling lucky”, scARE will return the first result. Link in result table will lead to RTE information page (section 4.a).

                                             ii.     Search by gene

In this option, scARE will search genes matching keyword. Result will be a table of matched genes and RTEs found within each matched gene. By checking “I’m feeling lucky”, scARE will return the first result. Link in result table will lead to RTE information page (section 4.a).

                                            iii.     Search by disease

In this option, scARE will search disease field of all dataset records and return matched list of datasets included in scARE. Link in result table will lead to dataset information page (section 4.b).

                                            iv.     Search by cell

                                              v.     Search by dataset

scARE will return list of datasets with scARE ID matching keywords. Link in result table will lead to RTE information page (section 4.b).

b.     Search multiple terms:

This option allows users to search for records in our database match all or any of several keywords. Returned results are similar with search single term.

2.     Browse options

a.     Browse by RTE

We provide 3 views for browsing by RTE: treemap view, tree view and table view. In treemap view, different classes/families/RTEs are represented as tiles and click on tile can expand each class/family or redirect to RTE information page (section 4.a). Numbers below each name is the total loci number of each class/family/RTE.

For tree view, class/family/RTE are organized as tree and nodes can be expanded or redirect to RTE information page (section 4.a).

Table view presents all RTEs as rows in a table with loci number of each RTE.  

b.     Browse by cell type

scARE provides options for browsing cell types identified in included samples. By selecting cell types shown, it will redirect to cell type information page (section 4.c).

c.     Browse by disease

scARE provides options to browse by all 3 diseases in our database. This page included basic information about each of Alzheimer’s disease, Parkinson’s disease and multiple sclerosis.

By clicking “View All AD Dataset” (or any of other two diseases), a table listing all datasets containing AD samples will popup and allow users to choose specific dataset to view detail (section 4.b).

d.     Browse by dataset

From this page, a list of all datasets in scARE will show and allow users to choose the specific dataset to view detail (section 4.b).

3.     Discover RTEs

a.     Most expressed RTEs.

This section will allow users to view most expressed RTEs in selected conditions. For example, the top 10 most expressed RTEs in excitatory neurons of AD samples.

scARE also provides table view of results, allowing users to gather more information.

b.     Most differential expressed RTEs - disease v.s. control

scARE compared expression of RTEs in disease v.s. control manner and allow users to find the most differential expressed RTEs in chosen condition.

c.     Most differential expressed RTES – cell v.s. cell

scARE also compared expression of RTEs in cell v.s. cell manner, allowing users to find most differential expressed RTEs between cell types.

d.     Co-expression network

We performed co-expression analysis for RTEs and genes in AD samples in manner of both disease v.s. control and cell v.s. cell. We then built the co expression network for the comparison. Clicking on names of RTEs (rectangles) will redirect to RTE information page (section 4.a)

                                               i.     Dataset information and cell number statistics

This section contains basic information of each dataset, allowing users to trace back to original publication. Cells detected in each dataset is also presented.

                                             ii.     Cell clustering and gene expression

Left panel shows the cell clustering, cell type annotation for all cells in this dataset while right panel shows expression of selected gene.
